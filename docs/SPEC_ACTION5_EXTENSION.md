# 📡 Action 5 — нічого розширювати не треба

## Контекст

Спочатку думав що треба розширювати API через `includeAll` прапор. Виправлено:

**Реально:** менеджери присутні у 1С. Просто Action 5 на сьогодні передає не всіх — внутрішня логіка трансмісії пропускає менеджерів деяких підрозділів (Колл-центр, Адасса, Чугуй=Полтава, Хайленко=Чернівці).

Користувач коректує це на стороні 1С. Без зміни API нашого боку.

## Що робимо ми

Нічого. Поточний код буде працювати як тільки 1С почне передавати повний список менеджерів. Адаптер у `src/lib/onec-adapters.ts` нічого не міняти.

## Як перевірити коли все буде готово

```bash
node scripts/diag-divisions.mjs
```

У секції `Action 5: getRegionData` має з'явитись 13 регіонів (зараз 11 — без Колл-центру, Адасси, Полтави, Чернівців).
