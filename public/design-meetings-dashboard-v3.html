Що змінилось vs v2

  • Картка ~25% нижче. Padding 14px (було 18), gap 10px (було 12), date-stamp на кожній картці видалено
  • Мета візиту як subtitle одразу під назвою клієнта (не floating-бейдж посередині). Природній порядок: хто → що → куди
  • «Локація зафіксована» інлайн біля адреси — не плаває окремо. Розділено вертикальною лінією від адреси, видно одразу що це pro адресу
  • Failed sync → ліва акцентна смуга на картці + малий warning-індикатор поряд із часом. Раніше floating-бейдж «Не синхр.» розривав композицію
  • Групування по днях — «Сьогодні · понеділок, 02 червня» як заголовок секції замість стіріптіза на кожній картці. Прогрес дня справа («1 завершено · 2 у роботі · 3 попереду»)
  • Зберіг: 0 емоджі, кнопки 44px+, mobile reflow з повноширинним primary на ≤480px
